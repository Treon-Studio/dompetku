Bagaimana QRIS akan disimpan?

Karena kita menggunakan Turso / SQLite lokal, penyimpanan gambar QRIS memiliki beberapa pilihan.

A

Menggunakan URL gambar eksternal

User memasukkan URL gambar QRIS mereka (misal dari hosting eksternal seperti Imgur/PostImages).

B

Simpan sebagai Base64 di database (Recommended)

User mengunggah file gambar langsung di aplikasi, lalu dikonversi dan disimpan sebagai string Base64 di kolom `qris_image` (Tipe data TEXT). Sangat praktis untuk ukuran QRIS yang kecil.
