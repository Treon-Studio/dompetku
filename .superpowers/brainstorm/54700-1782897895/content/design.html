Desain Fitur: Rekening Saya (Payment Accounts)

1. Model Data (SQLite / Drizzle)

export const payment_accounts = sqliteTable("payment_accounts", {
  id: text("id").primaryKey().$defaultFn(() => crypto.randomUUID()),
  bank_name: text("bank_name").notNull(), // Misal: BCA, Mandiri, GoPay
  account_number: text("account_number").notNull(), // Nomor rekening/HP
  account_holder: text("account_holder").notNull(), // Nama pemilik
  qris_image: text("qris_image"), // Gambar QRIS dalam format Base64
  user_id: text("user_id").notNull().references(() => users.id, { onDelete: "cascade" }),
  created_at: text("created_at").default(sql`CURRENT_TIMESTAMP`),
  updated_at: text("updated_at").default(sql`CURRENT_TIMESTAMP`),
});

2. Alur Pengguna (User Flow)

  • User mengklik menu "Rekening Saya" di Sidebar.
  • Halaman menampilkan daftar rekening yang sudah ditambahkan dalam bentuk kartu/list.
  • Ada tombol "Salin Nomor" (disertai toast sukses) dan tombol "Lihat QRIS" yang membuka dialog berisi gambar QRIS untuk di-scan.
  • Ada tombol "Tambah Rekening" yang membuka form input (Nama Bank/Dompet, Nomor, Nama Pemilik, dan Unggah File QRIS).
  • User bisa mengedit atau menghapus rekening yang ada.
